Path finder

Path Finder is a web application that allows users to visualize paths on a canvas grid. Users can interactively draw paths using the keyboard arrow keys, starting from the center of the grid. Once a path is drawn, users have the option to generate the shortest path or reset the drawn paths.

This project demonstrates a simple interactive path visualization with real-time keyboard input and pathfinding features.

  1. Click anywhere on the canvas to focus.
  2. Use the keyboard arrow keys to draw a path on the grid.
    • Each arrow key press moves one step in the corresponding direction.
  3. Once a path is drawn, you can:
    • Click the "Draw Shortest Path" button to calculate and visualize the shortest path from the starting point.
    • Click the "Reset Paths" button to clear the canvas and start over.
